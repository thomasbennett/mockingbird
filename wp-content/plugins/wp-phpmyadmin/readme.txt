This plugin lets you access phpMyAdmin from your WordPress administraion backend

== Description ==

WordPress functions as a wrapper for phpMyAdmin, so you have one thing less to log on.

== Installation ==

1. Download, unzip and upload to your WordPress plugins directory
2. activate the plugin within you WordPress Administration Backend
4. go to Manage > phpMyAdmin
